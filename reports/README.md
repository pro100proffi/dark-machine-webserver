# Отчет по проектной практике

## Введение

Проектная практика является важной частью образовательного процесса, так как позволяет студентам применить полученные теоретические знания на практике.

В рамках практики была проведена работа над концепцией инди-игры «Dark Machine».

---

# 1. Общая информация о проекте

## Название проекта

Dark Machine

## Цели проекта

Разработка концепта инди-игры в жанре метроидвания.

## Задачи проекта

- создание игрового мира;
- разработка механик;
- создание визуального стиля;
- подготовка документации.

---

# 2. Информация об организации

## Организация

Московский Политехнический Университет

## Деятельность

Университет занимается подготовкой специалистов в области IT, инженерии, дизайна и цифровых технологий.

---

# 3. Описание задания

В ходе практики необходимо было:

- освоить Git;
- изучить Markdown;
- создать статический сайт;
- оформить документацию проекта;
- подготовить итоговый отчет.

---

# 4. Полученные результаты

В ходе работы были:

- оформлены документы проекта;
- создан Git-репозиторий;
- разработан сайт проекта;
- подготовлена проектная документация;
- изучены основы командной разработки.

---

# Заключение

Практика позволила получить навыки работы с Git, Markdown, HTML/CSS и проектной документацией.

Полученные знания будут полезны для дальнейшей разработки IT- и GameDev-проектов.
