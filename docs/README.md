# О проекте

## Dark Machine

Dark Machine — это сюжетная пиксельная метроидвания в мире далекого постапокалиптического будущего, где человечество потеряло контроль над технологиями, а власть принадлежит машинам.

Игроку предстоит исследовать разрушенный мир, изучать тайны прошлого и принимать решения, влияющие на стиль прохождения и развитие персонажа.

---

## Особенности проекта

- Атмосферный 2D мир
- Пиксельная графика
- Система исследования
- Нелинейное развитие персонажа
- Два взаимоисключающих пути развития
- Уникальный визуальный стиль

---

## Игровой движок

Проект разрабатывается на движке **Godot 4**.

---

## Цели разработки

Основная цель проекта — получение практического опыта разработки игр и создание качественного концепта инди-игры.
