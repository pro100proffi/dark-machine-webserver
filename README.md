# Dark Machine Web Server

Учебный веб-проект, реализованный на собственном простом HTTP-сервере на Python.

Проект представляет собой сайт для концепта инди-игры **Dark Machine** — пиксельной метроидвании в мрачном постапокалиптическом мире, где власть принадлежит машинам, а человек становится частью производственной системы.

Сайт содержит основные разделы проекта, описание концепции, список участников, журнал разработки, ресурсы, а также использует CGI-скрипт для генерации серверного блока с таймером до дедлайна.

---

## О проекте

**Dark Machine** — концепт одиночной сюжетной инди-игры в жанре пиксельной метроидвании.

Действие игры происходит внутри гигантского завода, созданного машинами. Люди в этом мире больше не воспринимаются как личности — их выращивают в капсулах и используют как биологический ресурс.

Главный герой — один из таких людей. Из-за сбоя в системе его капсула выходит из строя, и он получает шанс выбраться из нижних уровней завода.

Основная идея проекта строится вокруг выбора между двумя путями:

- сохранение человеческой природы;
- механические модификации и постепенная потеря человечности.

---

## Функциональность сайта

Сайт включает следующие страницы:

- **Главная** — аннотация проекта, анимация и таймер до 6 июня;
- **О проекте** — подробное описание концепции, мира и геймплея;
- **Участники** — список участников проекта;
- **Журнал** — заготовки постов о ходе разработки;
- **Ресурсы** — ссылки на полезные материалы и место для репозитория.

---

## Особенности реализации

Проект не использует готовые веб-фреймворки.

Сервер написан вручную на Python и поддерживает:

- обработку HTTP-запросов;
- отдачу HTML-страниц;
- отдачу CSS и JavaScript-файлов;
- отдачу статических ресурсов;
- обработку несуществующих страниц через `404 Not Found`;
- запуск CGI-скриптов;
- генерацию серверного блока с текущим временем и расчётом до дедлайна.

---

## Используемые технологии

- **Python** — реализация HTTP-сервера;
- **Socket** — работа с TCP-соединениями;
- **Subprocess** — запуск CGI-скриптов;
- **HTML** — структура страниц;
- **CSS** — визуальное оформление и анимации;
- **JavaScript** — таймер обратного отсчёта и загрузка CGI-блока;
- **CGI** — серверная генерация дополнительного блока на главной странице.
