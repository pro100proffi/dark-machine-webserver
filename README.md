# Dark Machine

## О проекте

**Dark Machine** — пиксельная 2D метроидвания в сеттинге постапокалиптического будущего, где власть полностью принадлежит машинам.

Проект представляет собой одиночную сюжетную инди-игру с упором на исследование мира, атмосферу и развитие персонажа.

- Жанр: Pixel Metroidvania
- Формат: Singleplayer Indie Game
- Камера: 2D Side View
- Продолжительность: 5–10 часов
- Игровой движок: Godot 4

---

## Цель проекта

Цель проекта — разработать игровую концепцию, которая станет практикой для участников команды и сможет перерасти в полноценный коммерческий релиз.

---

## Основные задачи

- разработка оригинальной игровой концепции;
- создание игрового мира;
- проектирование боевой системы;
- разработка системы исследования;
- создание системы развития персонажа;
- разработка визуального стиля;
- подготовка концепт-документации.
