## Как запустить проект

Для запуска проекта нужен установленный Python 3.

Проверить установленную версию Python можно командой:

```bash
python --version
```

Если Python установлен правильно, терминал выведет примерно такой результат:

```text
Python 3.13.0
```

---

### 1. Склонировать репозиторий

```bash
git clone https://github.com/pro100proffi/dark-machine-webserver.git
```


---

### 2. Перейти в папку проекта

```bash
cd dark-machine-webserver
```

---

### 3. Запустить сервер

```bash
python main.py
```

После запуска в терминале должно появиться сообщение:

```text
Server running on http://127.0.0.1:8080
```

Это означает, что локальный сервер успешно запущен и готов принимать запросы от браузера.

---

### 4. Открыть сайт в браузере

После запуска сервера нужно открыть браузер и перейти по адресу:

```text
http://localhost:8080
```

Также можно использовать адрес:

```text
http://127.0.0.1:8080
```

## Важно

Проект запускается локально и по умолчанию работает на порту `8080`.

Если порт `8080` уже занят другой программой, можно изменить порт в файле `main.py`.

Нужно найти строку:

```python
PORT = 8080
```

и заменить, например, на:

```python
PORT = 8000
```

После этого сайт нужно будет открывать по новому адресу:

```text
http://localhost:8000
```

---

## Завершение работы сервера

Чтобы остановить сервер, нужно вернуться в терминал, где он был запущен, и нажать:

```text
Ctrl + C
```

После этого сервер остановится.
